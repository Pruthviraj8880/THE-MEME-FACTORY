# 🎭 Meme Explorer & Generator

## 📌 Project Overview

Meme Explorer & Generator is a web application that allows users to browse popular memes and (in future updates) create their own memes using templates.

## 🌐 API Used

This project uses the Imgflip API:
https://api.imgflip.com/get_memes

The API provides a list of popular meme templates including their images and names.

## 🎯 Features

* Browse trending memes
* Search memes by name
* Responsive design for mobile, tablet, and desktop

## 🚀 Future Enhancements

* Filter and sort memes
* Meme generator (add custom text)
* Download memes
* Dark mode

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript (Fetch API)

## ⚙️ How to Run the Project

1. Clone the repository:
   git clone https://github.com/your-username/meme-explorer.git

2. Open the project folder

3. Open index.html in your browser
